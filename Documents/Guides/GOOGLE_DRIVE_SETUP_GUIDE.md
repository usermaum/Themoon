# 구글 드라이브 연동을 위한 설정 가이드

본 프로젝트의 이미지 업로드 기능을 사용하기 위해 **Google Service Account(서비스 계정)** 설정이 필요합니다.
아래 순서대로 따라하시면 약 5분 정도 소요됩니다.

---

## 1단계: 구글 클라우드 프로젝트 생성

1. [Google Cloud Console](https://console.cloud.google.com/)에 접속합니다. (구글 로그인 필요)
2. 상단 메뉴의 **"프로젝트 선택"** (또는 현재 프로젝트 이름) 클릭 -> **"새 프로젝트"** 클릭.
3. **프로젝트 이름**을 입력합니다 (예: `TheMoon-Drive-Upload`).
4. **"만들기"** 버튼 클릭 후, 알림창에서 **"프로젝트 선택"**을 눌러 해당 프로젝트로 이동합니다.

## 2단계: Google Drive API 활성화

1. 좌측 메뉴 상단 **[햄버거 버튼 ☰]** -> **[API 및 서비스]** -> **[라이브러리]** 선택.
2. 검색창에 **"Google Drive API"** 입력 후 선택.
3. **"사용"** 버튼 클릭.

## 3단계: 서비스 계정 생성 및 키 발급

1. 좌측 메뉴 **[API 및 서비스]** -> **[사용자 인증 정보]** 선택.
2. 상단 **"+ 사용자 인증 정보 만들기"** -> **"서비스 계정"** 선택.
3. **서비스 계정 세부정보 입력**:
    *   **서비스 계정 이름**: `drive-uploader` (원하는 이름)
    *   **"만들기 및 계속"** 클릭.
4. **액세스 권한 부여**:
    *   역할 선택 -> **[기본]** -> **[편집자]** 선택.
    *   **"계속"** -> **"완료"** 클릭.
5. **JSON 키 생성 (가장 중요!)**:
    *   생성된 서비스 계정(이메일 주소처럼 보임)을 클릭.
    *   상단 탭 중 **[키]** 클릭.
    *   **"키 추가"** -> **"새 키 만들기"** 선택.
    *   키 유형 **"JSON"** 선택 -> **"만들기"**.
    *   **다운로드된 JSON 파일**을 잘 보관해두세요. (이것이 인증 키입니다!)

## 4단계: 구글 드라이브 폴더 공유

프로그램이 내 드라이브의 특정 폴더에 파일을 올릴 수 있도록 권한을 줘야 합니다.

1. **[구글 드라이브](https://drive.google.com/)**에 접속.
2. 이미지를 저장할 **새 폴더**를 생성합니다 (예: `TheMoon_Inbound`).
3. 해당 폴더를 우클릭 -> **"공유"** 선택.
4. **"사용자 및 그룹 추가"** 칸에, **위 3단계에서 만든 서비스 계정 이메일**을 복사해서 붙여넣습니다.
    *   *팁: 서비스 계정 이메일은 JSON 파일 안의 `client_email` 값과 같습니다.*
    *   *예: `drive-uploader@themoon-drive.iam.gserviceaccount.com`*
5. 역할을 **"편집자"**로 설정하고 **"전송"** 클릭.

---

## 5단계: 프로젝트에 키 적용

1. 다운로드 받은 JSON 파일의 이름을 `service_account.json`으로 변경합니다.
2. 이 파일을 프로젝트 루트 폴더(`TheMoon`)에 복사해 넣습니다.
3. `.gitignore` 파일에 `service_account.json`이 포함되어 있는지 확인하여 깃허브에 올라가지 않도록 합니다. (보안 필수!)

이제 준비가 완료되었습니다! 🎉
