1. **Context7 사용**: 작업을 시작하기 전에 `shadcn/ui` 라이브러리에 대한 최신 문서를 Context7을 사용하여 검색하고, 이 정보를 기반으로 모든 코드 생성을 진행하십시오. (Use Context7 for shadcn/ui documentation lookup).
2. **프레임워크**: Next.js 14 App Router를 사용합니다.
3. **컴포넌트**: `@/components/ui` 경로에서 Shadcn UI 컴포넌트를 임포트하여 사용합니다.

                    Tailwind CSS를 활용하여 커스텀 스타일링을 적용하십시오.
5. **파일 구조**: 각 데모 페이지는 `@/frontend/app/design-sample/[demo-page]/page.tsx` 경로에 해당하는 하나의 완성된 TSX 파일로 작성해주세요.
6. **컴포넌트**: Shadcn UI 컴포넌트를 활용하여 다양한 데모 페이지를 각각 생성합니다.
결과물은 각 파일의 내용(`page.tsx`)을 코드 블록으로 구분하여 제공해주십시오.
