## 📋 개요

### 목적

웹 애플리케이션의 사용자 경험(UX) 향상을 위해 페이지 전환, 카드, 히어로 섹션 등에 **미니멀 스타일의 자연스러운 애니메이션**을 구현합니다.

### 요구사항

- **범위**: 모든 주요 페이지 및 컴포넌트
- **스타일**: 미니멀 (간결하고 자연스러운 애니메이션)
- **라이브러리**: Framer Motion (기존 설치됨)

## 🎨 애니메이션 스펙

### 공통 설정값

| 속성 | 값 | 설명 |
|------|-----|------|
| `duration` | 0.3s ~ 0.6s | 애니메이션 지속 시간 |
| `ease` | `easeOut` | 감속 이징 커브 |
| `y offset` | 20px | Y축 이동 거리 |
| `scale` | 0.95 ~ 1.1 | 스케일 변화 범위 |
| `stagger delay` | 0.05s ~ 0.1s | 순차 애니메이션 간격 |

### Framer Motion Variants

```typescript
// 기본 페이드인
fadeInVariants = {
    hidden: { opacity: 0 },
    visible: { opacity: 1 }
}

// 페이드인 + 위로 이동
fadeInUpVariants = {
    hidden: { opacity: 0, y: 20 },
    visible: { opacity: 1, y: 0 }
}

// 스케일 인
scaleInVariants = {
    hidden: { opacity: 0, scale: 0.95 },
    visible: { opacity: 1, scale: 1 }
}
```

## 📊 성능 고려사항

### 최적화 적용

- **GPU 가속**: `transform` 속성만 사용하여 GPU 렌더링 활용
- **will-change 자동 관리**: Framer Motion이 자동으로 처리
- **적정 duration**: 0.3~0.6초로 부드러우면서도 빠른 느낌 유지
- **stagger 최소화**: 0.05초로 대기 시간 최소화

### 권장사항

- 많은 카드(20개 이상)가 있는 경우 stagger delay를 0.03초로 줄일 것 고려
- 저사양 기기에서 테스트 필요
